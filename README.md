
# 심심북스 쿠폰 도메인

## 1. 프로젝트 소개
`심심북스`: https://github.com/nhnacademy-be8-simsimhe  

`심심북스`는 Java Spring 기반의 온라인 서점 서비스입니다.  
`심심북스 쿠폰 도메인` 프로젝트는 원본 프로젝트인 `심심북스`에서 제가 담당했던 쿠폰 도메인을 별도의 프로젝트로 분리해 성능을 고도화했습니다.  
정률 • 정액 할인, 생일 쿠폰 발급, 책•카테고리•전체 쿠폰 기능 등을 제공합니다.
